## 🖼️ Recommended Image Specifications

### Gallery Images
- **Size**: 800x600px minimum
- **Format**: JPG or PNG
- **Quality**: High quality, optimized for web
- **Content**: Showcase different areas of your resort

### Room Images
- **Size**: 800x600px minimum
- **Format**: JPG
- **Quality**: High resolution, well-lit photos
- **Content**: Show actual rooms from best angles

### Hero Background
- **Size**: 1920x1080px (16:9 ratio)
- **Format**: JPG
- **Content**: Stunning resort view or beach scene

## 🔄 How to Replace Images

1. **Prepare your images** with the recommended specifications
2. **Rename your images** to match the current filenames exactly
3. **Replace files** in the appropriate folders
4. **Test the website** to ensure images load properly

## 💡 Photography Tips

### Gallery Photos
- Take photos during golden hour (sunrise/sunset) for best lighting
- Include variety: wide shots, details, people enjoying facilities
- Show different seasons if applicable

### Room Photos
- Stage rooms professionally before photographing
- Use natural lighting when possible
- Take multiple angles of each room type
- Include bathroom shots for luxury rooms

### General Tips
- Use a professional camera or high-quality smartphone
- Ensure photos are sharp and well-exposed
- Consider hiring a professional resort photographer
- Update photos seasonally if your resort changes appearance

## 🛠️ Technical Notes

- Images are referenced in both HTML and JavaScript files
- Keep the same filenames to avoid updating code
- Optimize images for web (compress without losing quality)
- Consider using WebP format for better performance (update code accordingly)

## 🎨 Image Optimization

Before uploading your images:
1. **Resize** to recommended dimensions
2. **Compress** using tools like TinyPNG or ImageOptim
3. **Rename** to match existing filenames
4. **Test** loading times on different devices
